# Descrição do produto

Você é um Coffee Lover? Se sim, esse é o kit perfeito pra você que quer saborear um cafezinho com sequilhos deliciosos. A combinação desses sequilhos com café é simplesmente incrível!

Além disso, esse kit vem com uma caneca exclusiva, com arte criada especialmente para Sequilhato, pelo artista Sérgio Bordim.

É também uma excelente opção para presentear pessoas especiais, afinal sequilho com café é o mesmo que um abraço apertado.

O kit é composto por:

  • 1 pouche Sequilho de Canela com 200g
  • ·         1 pouche de Sequilho Amanteigado com 200g
  • ·         1 pouche de Sequilho de Goma 200g
  • ·         1 caneca exclusiva
